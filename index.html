My Future Plans

After high school, I plan to study psychology. Understanding the mind and behavior connects with concepts like data, analysis, and how humans process information.

Unit 1: Data Concepts

I’ve learned that a bit is a binary digit (0 or 1) and a byte is a group of 8 bits. Data can suffer roundoff errors when not enough bits are used, and analog data changes smoothly over time. Compression can be lossless (perfect recovery) or lossy (some data lost). Metadata provides information about other data, like details about a file or database record.

Unit 2: Computer Systems & Networks

A computing device is any physical machine that can run programs. Devices form a computer network to share data. Data travels along a path and follows a protocol of communication rules. The Internet is fault tolerant and designed for scalability. Key ideas include bandwidth, unique IP addresses, and the use of packets routed across networks using TCP, UDP, and the Domain Name System (DNS).
