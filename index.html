Unit 1: Data Concepts

A bit is a binary digit (0 or 1) and a byte is a group of 8 bits. Data can suffer roundoff errors when not enough bits are used, and analog data changes smoothly over time. Compression can be lossless or lossy. Metadata provides information about other data.

Unit 2: Computer Systems & Networks

A computing device is any physical machine that can run programs. Devices form a computer network to share data. Data travels along a path and follows a protocol. The Internet is fault tolerant and designed for scalability.
